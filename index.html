⚙️ Setup Integrasi Aplikasi

🚨 PENTING: Cara Deploy Google Apps Script

Pastikan Anda telah mendeploy script penanganan ini ke akun Google Sheet Anda dengan benar:

  1. Buka dokumen Spreadsheet Google Anda.
  2. Akses menu Ekstensi / Extensions → Apps Script.
  3. Hapus berkas kode bawaan lama, dan rekatkan (paste) seluruh script di bawah ini.
  4. Klik dropdown menu fungsi di bagian atas script, pilih setupPermissions kemudian klik tombol Run untuk mengaktifkan izin Drive.
  5. Klik Deploy → New Deployment. Pilih opsi Web App.
  6. Atur hak akses: Who has access: Anyone. Lalu salin URL Web App yang dihasilkan.
